# 📁 Markdown Editor v1.2 — File Management

## Goal
Upgrade dari single-document jadi multi-document app dengan file explorer,
tab system, dan kemampuan buka/simpan file .md asli.

---

## New Features
- Multiple tabs (buka beberapa dokumen sekaligus)
- Save as .md file (download file markdown)
- Open .md file (drag & drop atau file picker)
- Recent files (list file terakhir dari localStorage)
- Rename & delete dokumen
- Sidebar file explorer

## Feature Breakdown

### 1. Multiple Tabs
- TabBar.jsx tampilkan semua dokumen yang sedang dibuka
- Klik tab → ganti activeDocId
- Tombol × di tiap tab → tutup tab (bukan hapus dokumen)
- Tombol [+ New] → buat dokumen baru + langsung buka di tab baru
- Jika semua tab ditutup → tampilkan empty state

### 2. Save as .md File
- Tombol [Save .md] di footer
- Trigger download file dengan nama `{title}.md`
- Gunakan Blob + URL.createObjectURL
```js
// exportMd.js
export const saveAsMd = (content, filename) => {
  const blob = new Blob([content], { type: "text/markdown" })
  const url = URL.createObjectURL(blob)
  const a = document.createElement("a")
  a.href = url
  a.download = `${filename}.md`
  a.click()
  URL.revokeObjectURL(url)
}
```

### 3. Open .md File
- Tombol [Open .md] → buka file picker (input type=file, accept=".md,.txt")
- Drag & drop ke area editor → baca file
- File dibaca dengan FileReader API
- Setelah dibaca → buat dokumen baru dari konten file tersebut
```js
// importMd.js
export const readMdFile = (file) => {
  return new Promise((resolve, reject) => {
    const reader = new FileReader()
    reader.onload = (e) => resolve(e.target.result)
    reader.onerror = reject
    reader.readAsText(file)
  })
}
```

### 4. Sidebar File Explorer
- List semua dokumen yang tersimpan di localStorage
- Klik dokumen → buka di tab baru (atau switch ke tab yang sudah ada)
- Hover dokumen → muncul tombol [✏️ Rename] dan [🗑️ Delete]
- Rename inline (input text langsung di sidebar)
- Delete dengan konfirmasi sederhana (window.confirm atau modal kecil)
- Sort by: updatedAt terbaru di atas

### 5. Recent Files
- Section terpisah di bawah sidebar
- Simpan max 10 file terakhir yang dibuka/diedit
- Tiap item tampilkan: title + waktu relatif (e.g. "2 hours ago")
- Klik → buka dokumen tersebut

## Notes untuk Agent
- Semua storage masih pakai localStorage, belum butuh backend
- Gunakan `uuid` dari package `uuid` bukan `crypto.randomUUID()` untuk kompatibilitas lebih luas
- DropZone: prevent default drag event di `dragover` agar browser tidak buka file sendiri
- Saat import file .md, judul dokumen = nama file tanpa ekstensi
- Tab yang aktif saat ini harus persist ke localStorage supaya tidak reset saat refresh
- Jika dokumen dihapus sementara tabnya masih terbuka, tutup tab tersebut otomatis
- Gunakan `updatedAt` untuk sorting dokumen di sidebar (terbaru di atas)
