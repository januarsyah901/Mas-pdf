# 📝 Markdown Editor — v1.1 Planning (Polish & UX)

## Goal
Meningkatkan kenyamanan penggunaan dari v1.0 tanpa menambah fitur besar.
Fokus: feel lebih smooth, responsif, dan profesional.

---

## New Features

### 1. Resizable Panels
- Tambah drag handle di border tengah antara editor dan preview
- Gunakan library `react-resizable-panels`
- Simpan ukuran panel terakhir ke localStorage

### 2. Focus Mode
- Tombol toggle di toolbar atau shortcut `Ctrl+Shift+F`
- Sembunyikan toolbar dan footer saat aktif
- Hanya tampilkan editor full-width
- ESC atau toggle ulang untuk keluar focus mode

### 3. Word & Character Count (Live)
- Tampil di footer kanan
- Hitung: words, characters (with & without spaces)
- Update real-time saat user mengetik
- Format: `Words: 120 | Chars: 643`

### 4. Scroll Sync
- Editor dan preview scroll secara proporsional bersamaan
- Gunakan `onScroll` event di editor, hitung rasio posisi, apply ke preview
- Bisa di-toggle on/off via tombol di footer

### 6. Custom PDF Options
- Modal sebelum export muncul dengan opsi:
  - Nama file (input text)
  - Paper size: A4 / Letter / A3
  - Margin: Normal / Narrow / Wide
  - Include / exclude halaman judul otomatis
- Simpan preferensi terakhir ke localStorage

### 7. Toast Notifications
- Library: `react-hot-toast` atau buat custom ringan
- Trigger toast saat:
  - Auto-save berhasil → `"Saved ✓"`
  - Export PDF selesai → `"PDF exported ✓"`
  - File di-load → `"Document loaded ✓"`
- Posisi: bottom-right
- Auto-dismiss: 2.5 detik

## Build Steps (untuk Agent)

1. Install dependencies baru:
   `npm install react-resizable-panels react-hot-toast`

2. Refactor `App.jsx`:
   - Wrap panel kiri & kanan dengan `react-resizable-panels`
   - Tambah state `focusMode` dari hook `useFocusMode`
   - Kondisikan render toolbar & footer berdasarkan `focusMode`

3. Buat `useScrollSync.js`:
   - Terima dua ref (editorRef, previewRef)
   - `onScroll` di editor → hitung scrollRatio → apply ke preview
   - Return: `{ syncEnabled, toggleSync, handleEditorScroll }`

5. Buat `useFocusMode.js`:
   - State boolean `isFocusMode`
   - Toggle function
   - Return `{ isFocusMode, toggleFocusMode }`

6. Buat `wordCount.js` utility:
   - `countWords(text)` → split by whitespace, filter empty
   - `countChars(text)` → `text.length`
   - `countCharsNoSpace(text)` → `text.replace(/\s/g, '').length`

7. Buat `Footer.jsx`:
   - Konsumsi word count dari `useMarkdown`
   - Tombol toggle scroll sync
   - Tombol buka Export Modal

8. Refactor `ExportButton.jsx` → `ExportModal.jsx`:
   - State: `filename`, `paperSize`, `margin`, `includeTitlePage`
   - Load preferensi dari localStorage saat modal dibuka
   - Simpan preferensi ke localStorage saat export
   - Pass opsi ke `exportPdf.js`

9. Update `exportPdf.js`:
   - Terima parameter `{ element, filename, paperSize, margin }`
   - Map margin ke nilai px: Normal=20mm, Narrow=10mm, Wide=30mm
   - Konfigurasi `html2pdf` dengan opsi tersebut

10. Buat `Toast.jsx` atau setup `react-hot-toast`:
    - Tambah `<Toaster />` di `App.jsx`
    - Panggil `toast.success("Saved ✓")` di lokasi yang tepat

11. Update `Toolbar.jsx`:
    - Tambah tombol focus mode (ikon fullscreen)
    - Tambah tombol toggle preview (hide/show preview panel)

13. Testing scroll sync di konten panjang
14. Testing export PDF dengan berbagai opsi
15. Polish animasi toast & transisi focus mode

## Notes untuk Agent
- `react-resizable-panels` butuh wrapper `PanelGroup` + `Panel` + `PanelResizeHandle`
- Scroll sync harus di-throttle dengan `requestAnimationFrame` supaya tidak lag
- Focus mode sebaiknya pakai CSS transition untuk smooth hide/show
- Modal export jangan pakai `<form>` tag — gunakan `div` + `onClick` handler
- Simpan semua preferensi user (panel size, scroll sync state, pdf options) ke localStorage dengan key prefix `mde_v11_`
