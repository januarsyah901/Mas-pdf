# Version 1.3.0 - 2026-04-01

## Added
- Table editor GUI - easily create markdown tables with alignment options
- Image paste/upload support - paste from clipboard or drag & drop images
- Math rendering with KaTeX - inline ($formula$) and block ($$formula$$) math support
- Mermaid diagram support - flowchart, sequence, gantt, pie chart diagrams
- Font selector - choose from 5 professional fonts (Poppins, Playfair Display, Lora, Fira Code, Times New Roman)
- Custom CSS panel - override preview styling with custom CSS
- Template starter system - quick start with README, Blog Post, Resume, Meeting Notes, and Technical Doc templates

## Fixed
- Improved image handling and validation
- Better error handling for mermaid diagrams
- Enhanced math rendering compatibility

## Changed
- Upgraded content editing capabilities with professional tools
- Enhanced preview rendering with KaTeX and Mermaid support
- Improved document starting experience with templates

## Removed
- (No features removed in this release)

## Notes
- All new features are fully client-side with localStorage persistence
- Font selections persist across sessions
- Custom CSS can be saved per document
- Template system available for new documents
- Image files limited to 2MB with validation warnings
- Full markdown table and diagram syntax supported
